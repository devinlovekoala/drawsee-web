# 电路实验文档库功能优化

## 修改内容

### 1. CircuitExperimentTask 页面
- 简化页面布局，采用单一的文件上传区域
- 移除了"或"的区分，使界面更加简洁
- 上传的文件直接保存到用户文档库中
- 添加了文件类型和大小的验证
- 右上角添加了"进入实验文档库"按钮

### 2. DocumentSelector 组件
- 修改为直接跳转到文档库页面的按钮
- 不再弹出选择对话框

### 3. DocumentUploader 组件
- 修改为直接跳转到文档上传页面的按钮
- 不再弹出上传对话框

### 4. DocumentUpload 页面
- 创建了新的文档上传页面
- 提供美观的文件上传界面
- 上传成功后跳转到文档库页面

### 5. DocumentAnalysis 页面
- 修复了"分析"按钮的崩溃问题
- 确保正确创建AI任务并跳转到会话页面
- 优化了错误处理和加载状态管理
- 移除了不必要的条件判断

## 用户体验改进
1. 简化了用户操作流程
2. 统一了文档上传和管理方式
3. 确保所有上传的文档都保存到用户文档库
4. 添加了更多的用户反馈和错误处理

## 技术改进
1. 修复了类型错误和组件间通信问题
2. 优化了文件上传和验证逻辑
3. 统一了页面跳转和状态管理方式
4. 提高了代码的可维护性和一致性

## BLOCK A · 模拟电路 · 无源元件补全
> 现有：R / C / L / GND，缺口大

### TASK-A01｜无源元件基础补全

新增以下 5 个高频教学元件，结构参照现有电阻(R)：

**1. 变压器 (Transformer)**
- 引脚：L1+ / L1- / L2+ / L2-
- 属性：`匝数比 n`（默认 1）、`初级电感 L1`（默认 1mH）
- SPICE：`K L1 L2 1` + 两个电感声明
- 符号：两组线圈 + 中间竖线（IEC标准）

**2. 电位器 (Potentiometer)**
- 引脚：P1 / P2 / Wiper（滑动端）
- 属性：`总阻值 R`（默认 10kΩ）、`滑动比例 ratio`（0~1，默认 0.5）
- SPICE：拆分为 R_upper = R×ratio，R_lower = R×(1-ratio)

**3. 开关 (Switch)**
- 类型属性：`SPST`（单刀单掷）
- 引脚：P1 / P2
- 属性：`初始状态`（断开/闭合）
- SPICE：闭合=0.001Ω电阻，断开=1GΩ电阻

**4. 互感 (Mutual Inductance)**
- 引脚：L1+ / L1- / L2+ / L2-
- 属性：`耦合系数 k`（0~1，默认 0.99）、`L1`、`L2`
- SPICE：`.model` K语句

**5. 节点标签 (Net Label)**
- 功能：同名标签视为导线直连（跨画布连接）
- 属性：`标签名`（可编辑文字）
- SPICE：输出为同一节点名称

**验收：** 变压器仿真，初级加10V正弦，匝数比=2时次级输出5V。

---

### TASK-A02｜无源元件寄生参数（改造现有R/C/L）

**改造目标**：在现有电阻、电容、电感属性面板中新增"寄生参数"折叠区块。

**电阻(R) 新增**：
- `寄生电感 LS`（默认 0，单位 nH）
- `寄生电容 CP`（默认 0，单位 pF）
- 启用后 SPICE 子电路：R串联LS，整体并联CP

**电容(C) 新增**：
- `ESR 等效串联电阻`（默认 0，单位 Ω）
- `ESL 等效串联电感`（默认 0，单位 nH）
- 启用后子电路：C + ESR + ESL 三者串联

**电感(L) 新增**：
- `DCR 直流电阻`（默认 0，单位 Ω）
- `SRF 自谐振频率`（可选，填入后自动反推并联电容值）

**UI要求**：默认折叠，标题"▶ 寄生参数（高级）"，不影响简单使用。

**验收：** 电容 ESR=1Ω 时，AC 扫描阻抗曲线在谐振点出现最小值后反弹。

## BLOCK B · 模拟电路 · 电源/信号源补全
> 现有：直流电压源(V)、直流电流源(I)、交流信号源(AC)

### TASK-B01｜信号源大幅扩充

在现有电源元件分类下，新增以下信号源（参照现有AC源结构）：

**1. 脉冲源 (VPULSE)**
- 属性：`V1`低电平、`V2`高电平、`延迟TD`、`上升时间TR`、`下降时间TF`、`脉宽PW`、`周期PER`
- SPICE：`V1 n+ n- PULSE(V1 V2 TD TR TF PW PER)`

**2. PWM源 (VPWM)**
- 属性：`频率FREQ`、`占空比DUTY(0~1)`、`高电平VHIGH`、`低电平VLOW`
- 底层自动换算为 PULSE 语句：`PW = DUTY/FREQ`，`PER = 1/FREQ`
- UI提示：这是对PULSE的封装，更直观

**3. 正弦源 (VSIN)**
- 属性：`直流偏置VO`、`幅值VA`、`频率FREQ`、`延迟TD`、`阻尼系数DF`
- SPICE：`SIN(VO VA FREQ TD DF)`

**4. 指数源 (VEXP)**
- 属性：`V1`、`V2`、`上升延迟TD1`、`上升时间常数TAU1`、`下降延迟TD2`、`下降时间常数TAU2`

**5. 调频源 (VSFFM)**
- 属性：`直流VO`、`幅值VA`、`载波频率FC`、`调制度MDI`、`调制频率FS`

**6. 受控电压源 VCVS**
- 引脚：控制端(+/-)、输出端(+/-)
- 属性：`增益E`（默认 1）
- SPICE：`E1 out+ out- ctrl+ ctrl- 1`

**7. 受控电流源 VCCS**
- 属性：`跨导GM`（默认 1）
- SPICE：`G1 ...`

**8. 三相电压源 (3-Phase)**
- 内部由3个VAC组成，自动设置0°/120°/240°相位
- 引脚：A / B / C / N(中性线)

**验收：** PWM源占空比=0.3、频率=1kHz，瞬态仿真波形高电平时间=0.3ms。

## BLOCK C · 模拟电路 · 半导体补全
> 现有：二极管(D)、NPN、PNP，严重不足

### TASK-C01｜二极管族扩充

在半导体分类下，参照现有二极管(D)新增：

**1. 稳压二极管 (Zener)**
- 符号：二极管 + 弯折阴极线
- 额外属性：`击穿电压 VZ`（默认 5.1V）、`膝点电流 IZT`（默认 5mA）
- SPICE：`.model ZENER D(BV=VZ IBV=IZT)`

**2. 发光二极管 (LED)**
- 符号：二极管 + 两条斜箭头
- 额外属性：`正向电压 VF`（默认 2V）、`颜色`（仅显示用：红/绿/黄/蓝/白）
- SPICE：`.model LED D(IS=... N=... RS=...)`

**3. 肖特基二极管 (Schottky)**
- 额外属性：`反向击穿电压 BV`、`正向压降 VF`（低于普通二极管，约0.3V）
- SPICE 参数：`IS`较大，`RS`较小

**4. 整流桥 (Bridge)**
- 符号：菱形4二极管桥
- 引脚：AC1 / AC2 / DC+ / DC-
- SPICE：`.subckt` 包含4个二极管

**5. TVS管 (TVS)**
- 类型属性：单向 / 双向
- 额外属性：`钳位电压 VC`、`击穿电压 VBR`
- 双向SPICE：两个反向并联Zener子电路

**验收：** Zener接5.1V击穿电压，反向电压加6V，仿真稳压输出≈5.1V。

---

### TASK-C02｜MOSFET 新增

新增4种场效应管，引脚和符号参照教材标准：

**1. NMOS 增强型**
- 引脚：Gate(G) / Drain(D) / Source(S) / Body(B，可选连接)
- 属性：`阈值电压 VTH`（默认 2V）、`跨导参数 KP`（默认 0.1A/V²）、`沟道调制 LAMBDA`（默认 0.01）
- SPICE：`.model NMOS_E NMOS(LEVEL=1 VTO=2 KP=0.1 LAMBDA=0.01)`

**2. PMOS 增强型**
- 同上，VTO 为负值（默认 -2V）

**3. N-JFET**
- 引脚：Gate / Drain / Source
- 属性：`夹断电压 VP`（默认 -4V）、`零偏漏电流 IDSS`（默认 10mA）

**4. P-JFET**
- 同上，VP 为正值

**验收：** NMOS 共源放大电路，VGS 扫描，输出 ID-VGS 转移特性曲线可见门限电压。

---

### TASK-C03｜功率半导体新增

**1. IGBT (N型)**
- 引脚：Gate / Collector / Emitter
- 符号：MOSFET符号 + BJT输出级组合
- SPICE：MOSFET + PNP 宏模型子电路
- 属性：`VGE(th)`栅极阈值（默认 4V）、`VCE(sat)`（默认 2V）

**2. SCR 晶闸管**
- 引脚：Anode(A) / Cathode(K) / Gate(G)
- SPICE：PNP + NPN 两管互补触发子电路
- 属性：`VTM`通态压降（默认 1.5V）、`IH`维持电流（默认 10mA）、`IGT`触发电流（默认 15mA）

**3. TRIAC 双向晶闸管**
- 引脚：MT1 / MT2 / Gate
- SPICE：两个反向SCR子电路

**验收：** SCR电路，门极触发脉冲后，仿真阳极-阴极导通，移除触发后电流高于IH时维持导通。

---

### TASK-C04｜BJT 模型精度提升（改造现有NPN/PNP）

在现有NPN/PNP属性面板中增加精度选项：

**新增"模型精度"下拉**（默认选"标准"）：
- `简化`：固定 β=100，IS=1e-14，适合快速拓扑验证
- `标准（推荐）`：补充 VAF、IKF、RC 等参数（NPN映射2N3904，PNP映射2N3906）
- `精确`：完整 Gummel-Poon 参数

**标准模型参数（直接硬编码为默认值）**：
```
NPN标准: IS=6.734f BF=416 VAF=74 IKF=66.78m RC=1 CJC=3.638p CJE=4.493p TF=301.2p
PNP标准: IS=1.41f BF=180 VAF=18.7 IKF=80m RC=0.6 CJC=9.728p CJE=8.063p
```

**验收：** NPN标准模型，仿真β值在150~400范围（不再是固定100），输出特性曲线可见Early效应斜率。

---

### TASK-C05｜运算放大器模型分级（改造现有运放）

在现有运算放大器属性面板新增"模型精度"选项：

**Level 1 理想（现有行为保留）**：无限增益、无限带宽

**Level 2 宏模型（新增，设为默认）**：
- 新增属性字段：
  - `GBW` 增益带宽积（默认 1MHz）
  - `SR` 压摆率（默认 0.5 V/μs）
  - `VOS` 输入失调电压（默认 1mV）
  - `VSAT` 输出饱和裕量（默认 1.5V，即输出最大摆至 VCC-1.5V）
  - `IOUT_MAX` 最大输出电流（默认 25mA）

**Level 3 芯片型号（新增）**：
- 下拉选择内置型号：`μA741 / LM358 / LM324 / TL071 / LM318 / OP07`
- 每个型号有对应的 `.subckt` 宏模型文件内置在资源中

**验收：** Level 2 运放 AC 扫描，在 GBW=1MHz 时，0dB 交叉频率=1MHz；大信号阶跃响应斜率受 SR 限制。

## BLOCK D · 数字电路 · 全面补全
> 现有极少：逻辑门(7种) + D触发器 + 数字I/O(3种)，缺口极大

### TASK-D01｜逻辑门补全

在现有7种逻辑门基础上，补充以下变体（同分类下）：

**新增元件**：
- `NAND 三输入` / `NAND 四输入`（现有NAND仅2输入则补充）
- `NOR 三输入` / `NOR 四输入`
- `AND 三输入` / `AND 四输入`
- `OR 三输入` / `OR 四输入`
- `XNOR`（若现有则确认，否则新增）
- `缓冲器 BUF`（三角形符号，无反相圆圈）
- `三态缓冲器 TRI`（引脚：IN / OE / OUT，OE=0时输出高阻）
- `史密特触发反相器`（符号：NOT + 内部滞回符号）

**对现有逻辑门的改造**：
- 所有逻辑门新增属性：`传播延迟 TPD`（默认 5ns）
- 数字仿真中，输出波形相对输入延迟 TPD 时间

**验收：** 三态缓冲器 OE=0 时，输出节点呈高阻（与其他驱动源可总线共存，互不冲突）。

---

### TASK-D02｜触发器全系补全

参照现有D触发器，新增：

**1. JK 触发器**
- 引脚：J / K / CLK / Q / Q̄ / PRE（异步置位）/ CLR（异步清零）
- 行为：J=1,K=0→置1；J=0,K=1→清零；J=K=1→翻转；J=K=0→保持

**2. T 触发器**
- 引脚：T / CLK / Q / Q̄ / CLR
- 行为：T=1时每个时钟翻转；T=0时保持

**3. SR 触发器**
- 引脚：S / R / CLK / Q / Q̄
- 行为：标准SR逻辑，S=R=1为禁止状态（仿真输出X）

**4. SR 锁存器（无时钟）**
- 引脚：S / R / Q / Q̄（电平触发，无CLK）

**5. D 锁存器（无时钟）**
- 引脚：D / EN / Q / Q̄（EN高电平时透明传输）

**对现有D触发器的改造**：
- 补充引脚：`PRE`（异步置位，低有效）/ `CLR`（异步清零，低有效）
- 补充属性：`触发边沿`（上升沿/下降沿，默认上升沿）
- 补充属性：`建立时间 TSETUP`（默认 3ns）/ `保持时间 THOLD`（默认 1ns）

**验收：** JK触发器 J=K=1，时钟连续输入，Q端输出二分频方波。

---

### TASK-D03｜计数器全系新增

**1. 4位二进制计数器**
- 引脚：CLK / CLR / Q[3:0] / RCO（进位输出）
- 属性：`计数方向`（加/减，默认加法）

**2. BCD 十进制计数器**
- 引脚：CLK / CLR / Q[3:0] / RCO
- 行为：计数到9后归零，RCO产生进位脉冲

**3. 可预置计数器（74161行为）**
- 引脚：CLK / CLR / LOAD / ENP / ENT / D[3:0] / Q[3:0] / RCO
- 属性：`模值 N`（2~16，默认16）

**4. 可逆计数器**
- 引脚：CLK / CLR / UP_DOWN / Q[3:0] / RCO / BCO

**5. 环形计数器**
- 引脚：CLK / CLR / Q[N-1:0]
- 属性：`位数`（默认4）

**验收：** BCD计数器，连续10个时钟脉冲，Q值完整经历0→9→0，RCO在9→0时脉冲一次。

### TASK-D04｜寄存器新增

**1. 4位并行寄存器**
- 引脚：CLK / D[3:0] / Q[3:0] / OE

**2. 8位并行寄存器**
- 同上，位宽8

**3. 串入并出 SIPO（74595行为）**
- 引脚：SER / SRCLK / RCLK / OE / SRCLR / Q[7:0] / QH'

**4. 并入串出 PISO**
- 引脚：CLK / LOAD / D[3:0] / Q(串行输出) / QN

**5. 通用移位寄存器（74194行为）**
- 引脚：S0 / S1 / CLK / CLR / DSR / DSL / D[3:0] / Q[3:0]
- 模式：S0S1=00保持，01右移，10左移，11并行置数

**验收：** 74595串行输入8位数据 10110010，8个SRCLK后RCLK打入，Q[7:0] = 10110010。

### TASK-D05｜组合逻辑模块新增

**1. 3-8 译码器（74138行为）**

**4. 8选1多路选择器（74151行为）**
- 引脚：D0~D7 / S0~S2 / E / Y / W

**5. 4位数值比较器（74HC85行为）**
- 引脚：A[3:0] / B[3:0] / IAGTB / IAEQB / IALTB / OAGTB / OAEQB / OALTB

**6. 4位全加器（74283行为）**
- 引脚：A[3:0] / B[3:0] / C0 / S[3:0] / C4

**7. 8位透明锁存器（74373行为）**
- 引脚：D[7:0] / Q[7:0] / OE / LE

**8. 8位总线收发器（74245行为）**
- 引脚：A[7:0] / B[7:0] / DIR / OE

**每个芯片必须**：属性面板内嵌功能表（折叠区块，标题"📋 功能表"），对照童诗白教材内容。

**验收：** 74138，G1=1,G2A=G2B=0，ABC=011时，仅Y3=0，其余Y为1。

### TASK-D06｜数字 I/O 补全

在现有"数字输入/输出/时钟源"基础上新增：

**1. 七段数码管**
- 引脚：a/b/c/d/e/f/g/DP（各段）/ COM（共阴/共阳可选）

**2. LED 指示灯（数字版）**

**3. 虚拟按钮**
- 仿真中可鼠标点击切换高低电平（交互式仿真）
- 属性：`默认状态`（低/高）、`模式`（自锁/点动）

**4. 总线 (Bus)**
- 功能：将多条数字导线聚合显示为粗线
- 属性：`位宽`（默认8）

**5. 总线分支 (Bus Entry)**
- 将总线中的单条信号引出

**验收：** 七段数码管连接 BCD 计数器，仿真中数码管依次显示 0~9。

### TASK-D07｜存储器元件新增

**1. ROM（可编程只读存储器）**
- 引脚：ADDR[4:0] / DATA[7:0] / OE
- 属性面板：内嵌32行 HEX 编辑器（可手填每个地址的数据）

**3. FIFO 缓冲器**

- 属性：`深度`（8/16/32，默认8）

**验收：** ROM 地址 0x03 预设值 0xAB，仿真读取该地址 DATA 输出 = 0xAB。

## 执行顺序（按教学价值排优先级）

```
第一批（基础教学必须，影响面最广）：
B01(信号源) → C01(二极管族) → C04(BJT精度) → C05(运放分级)
```

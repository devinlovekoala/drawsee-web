

# ⭐ 教材级 PDF文档 → 智能实验任务解析系统实现方案

下面是一个经常用于“实验课程自动理解”的成熟路线：

```
PDF → 结构解析 → 段落/标题提取 → 实验内容分类 → 分模块调用模型 → 结果汇总
```

------

# 1) 文档解析阶段（PDF → 结构化文本）

对于你上传的文档结构，很规律：
 **标题（实验编号） → 实验目的 → 实验设备 → 准备要求 → 实验步骤 → 表格 → 问题 → 实验报告要求**。

这非常适合通过 PDF 结构解析组件处理。

### 推荐技术栈（可任选）

| 组件             | 推荐工具                | 为什么适合电子实验文档            |
| ---------------- | ----------------------- | --------------------------------- |
| PDF 文本提取     | `pdfplumber`、`PyMuPDF` | 能保留表格、标题、段落结构        |
| 表格结构解析     | `camelot`, `tabula`     | 文档里有大量表格（表9.*.*）       |
| 标题层级识别     | 正则 + NLP 标题分类     | “实验1/2/3”、“一、二、三”格式固定 |
| 图示抽取（可选） | PyMuPDF + OCR           | 实验电路图可提取为图片            |

输出成为结构化数据：

```json
{
  "experiment_id": "实验 1",
  "sections": [
    {"title":"实验目的", "content":"..."},
    {"title":"实验设备", "content":"..."},
    {"title":"准备要求", ...},
    {"title":"实验内容", ...},
    {"title":"表格", ...},
    {"title":"开放问题", ...},
    …
  ]
}
```

------

# 2) 任务语义分类（非常关键）

对每一段内容进行语义分类，让大模型能“知道”段落属于哪种类型：

| 类型           | 示例                    | AI 作用                    |
| -------------- | ----------------------- | -------------------------- |
| **实验目标**   | 掌握 BJT 放大器测量方法 | 输出结构化总结             |
| **实验步骤**   | 调节 Rb 使 Vc = 6V      | 自动生成步骤解释、注意事项 |
| **电路图描述** | 图9.1.1                 | 生成电路组成讲解           |
| **表格任务**   | 表9.1.2 增益测量        | 转成 JSON、生成分析提示    |
| **开放问题**   | “如何防止失真？”        | 自动给答案                 |
| **注意事项**   | 波形失真产生原因        | 自动生成风险提示           |

技术方案：

- 直接用 LLM 做段落分类（准确率很高）。
- 或使用小模型（BERT/RoBERTa）做分类（可离线）。

------

# 3) 模块化的大模型调用（你的应用的核心）

你不想用 RAG，这完全没问题，因为这个文档属于**结构化教学类文档**，模型能直接处理。

下面是推荐的**模块化 prompt pipeline**：

------

## **模块 A：实验目的/设备/要求总结（通用 summarizer）**

输入：对应段落
 输出：结构化 JSON：

```json
{
  "objectives": [...],
  "equipment": [...],
  "preparation": [...]
}
```

------

## **模块 B：实验步骤解析（Instruction → 解释 + 风险 + 注意事项）**

输入：例如：

> （1）用万用表判断三极管极性。

模型输出：

- 步骤解释（为什么要这么做）
- 关键电路理论
- 常见错误
- 操作风险
- 实验意义

------

## **模块 C：电路原理分析（CE 放大器 / 积分器 / 微分器专用分析模型）**

这是你项目最重要的能力！
 可以基于通用 LLM，通过 prompt 设计成为**专业电路分析专家**：

输入：

> 图9.1.2 共射极放大电路，R2=51Ω…

输出：

- 电路功能总结
- 直流工作点推导
- 增益推导
- 输入/输出阻抗公式
- 为什么信号会衰减
- 典型波形应该长什么样

------

## **模块 D：表格理解与公式推导**

例如：

表9.1.2：空载时电压增益

模型可以：

- 解析表格 → 转成 JSON
- 自动计算增益（Vo / Vi）
- 与理论公式对比
- 给出误差来源

------

## **模块 E：问题回答（开放问题）**

例如：

> 什么会造成饱和失真和截止失真？

大模型可以自动根据电路工作区解释：

- 载流子饱和
- 放大区 → 截止区边界
- 输入信号过大造成集电极结饱和等

并能“引用文档上下文”但不是 RAG—而是已有段落。

------

## **模块 F：自动生成实验报告（非常强大）**

根据前面所有结果组合，自动生成：

- 结果表格
- 数据分析
- 波形预期
- 理论与实验对比
- 误差分析
- 开放问题回答

------

# 4) 整体系统架构（不使用向量数据库）

这就是适合你的最终架构：

```
PDF → PDF 解析器
      ↓
结构化段落
      ↓
段落分类器
      ↓
模块化 LLM 调用（A/B/C/D/E）
      ↓
结果聚合器（汇总、去重复、格式化）
      ↓
输出：实验解析 / 自动实验报告 / 关键要点
```

完全不需要向量库，也不需要检索数据库
